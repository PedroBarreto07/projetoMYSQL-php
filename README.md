# Projeto PHP com MySQL

Este projeto consiste em uma aplicação simples desenvolvida em PHP integrada com banco de dados MySQL, com o objetivo de praticar operações básicas de um sistema web.

## Funcionalidades
- Inserção de dados no banco de dados
- Listagem de registros
- Exclusão de dados
- Conexão com banco MySQL

## Tecnologias utilizadas
- PHP
- MySQL
- HTML
- XAMPP

## Estrutura do projeto
- `index.php` → Página principal
- `inserir.php` → Responsável por inserir dados
- `excluir.php` → Responsável por excluir registros
- `conexao.php` → Conexão com o banco de dados
